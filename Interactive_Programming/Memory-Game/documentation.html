Food Memory: Leap Motion Documentation

This project was inspired by a childhood game. I've already made games where the user has to collect coins and avoid enemies, and I have been fascinated by what is possible with the Leap Motion sensor, so I thought it would be fun to make a classic memory game where the user gets to use physical click motions. At first, I made the game using mouse clicks, which made it easy to convert the code to work with the Leap. The most challenging part was keeping track of how many tiles could be revealed at a time and for how long to display the images, so I had to use setTimeout a couple times. Additionally, I love food and minimalistic graphics, so I decided to use those as my tile images.
